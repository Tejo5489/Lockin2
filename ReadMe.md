LockIn | Weaponize Your Focus

Zero Distractions. Absolute Execution. LockIn is a brutally strict, private web application designed to track habits, manage deadlines, and use AI to monitor your flow state.

🚀 How it Works

100% Local & Private: All habit data, tasks, and analytics are stored strictly in your browser's local storage.

AI Neural Engine: Uses MediaPipe Face Mesh to invisibly track your head posture and eye movement to calculate a Live Focus Score.

Neo-Brutalist UI: A sleek, distraction-free interface built with Tailwind CSS.

🛠️ Deployment

This application is completely self-contained.
To host, simply deploy the index.html file to Vercel, Netlify, or GitHub Pages. No backend server or build step is required!
